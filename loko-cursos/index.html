Seja bem vindo ao Loko Cursos

O que é o Loko Cursos?

O Loko Cursos é a iniciativa de iniciar cursos gratuitos e particulares de maneira mais fácil e acessíve. Sendo que, os cursos gratuitos estão acessíveis no Youtube, enquanto os particulares podem ser pagos por aula, sendo assim, se não ter aula, não tem dívida.

Quais os cursos disponíveis?

O Loko Cursos vai cobrir 3 categorias de cursos, sendo eles:
computer

Informática

Informática básica à avançada, passando ali pelo o que é o computador até pacote de escritório.

design_services

Design Gráfico

Ilustração, marketing digital e animação

code

Programação

Programação Front e Back end com algumas linguagens, como Python, JS, Java, Dart, Kotlin, entre outras.
